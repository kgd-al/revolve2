OpenAI ES optimizer.

Based on: https://gist.github.com/karpathy/77fbb6a8dac5395f1b73e7a89300318d https://openai.com/blog/evolution-strategies/

async ainit_from_database(database: AsyncEngine, session: AsyncSession, db_id: DbId, rng: Random) → bool

Try to initialize this class async from a database.

Called when creating an instance using from_database.

Parameters

  • database – Database to use for this optimizer.

  • session – Session to use when loading and saving data to the database during initialization.

  • db_id – Unique identifier in the completely program specifically made for this optimizer.

  • rng – Random number generator used in the complete optimization process. Its state will be overwritten with the serialized state from the database.

Returns

True if the complete object could be deserialized from the database.

Raises

IncompatibleError – In case the database is not compatible with this class.
