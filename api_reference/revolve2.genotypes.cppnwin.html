crossover_v1(parent1: Genotype, parent2: Genotype, multineat_params: Parameters, rng: RNG, mate_average: bool, interspecies_crossover: bool) → Genotype

Perform crossover between two CPPNWIN genotypes.

Both genotypes will not be altered.

Parameters

  • parent1 – Genotype 1.

  • parent2 – Genotype 2.

  • multineat_params – Multineat parameters. See Multineat library.

  • rng – Random number generator.

  • mate_average – If the weights of matching connections between the parents should be averaged instead of choosing the value from one of the parents. See NEAT algorithm.

  • interspecies_crossover – TODO description. Choose False if you don’t know what this means. See Multineat library algorithm.

Returns

Genotype that is the result of crossover.

  • multineat_params – Multineat parameters. See Multineat library.

  • rng – Random number generator.

random_v1(innov_db: InnovationDatabase, rng: RNG, multineat_params: Parameters, output_activation_func: ActivationFunction, num_inputs: int, num_outputs: int, num_initial_mutations: int) → Genotype

Create a random CPPNWIN genotype.

A CPPNWIN network starts empty. A random network is created by mutating num_initial_mutations times.

Parameters

  • innov_db – Multineat innovation database. See Multineat library.

  • rng – Random number generator.

  • multineat_params – Multineat parameters. See Multineat library.

  • output_activation_func – Activation function for the output layer. See Multineat library.

  • num_inputs – Number of input for the network.

  • num_outputs – Number fo outputs for the network.

  • num_initial_mutations – The number of times to mutate to create a random network.

Returns

The created genotype.
