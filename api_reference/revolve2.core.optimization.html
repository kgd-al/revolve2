class Process

Bases: object

Provides asynchronous instantiation of the class that inherits from this.

The inherited class must implement the ainit_new and ainit_from_database functions, as specified in the comments below.

Instantiate either using new or from_database. In the latter case the inherited class must attempt to load its state from the provided database.

However the ainit functions cannot be made abstractmethod because the arguments are not known in advance(they could be retrieved using a decorator though), so that would violate lizkov’s substitution principle.
