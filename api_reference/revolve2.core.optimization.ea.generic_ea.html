class EAOptimizer

Bases: Process, Generic[Genotype, Fitness]

A generic optimizer implementation for evolutionary algorithms.

Inherit from this class and implement its abstract methods. See the Process parent class on how to make an instance of your implementation. You can run the optimization process using the run function.

Results will be saved every generation in the provided database.
